Project Overview

Room Raider is a Mixed Reality dodgeball game that transforms the player’s real environment into a dynamic gameplay arena using Meta Quest’s Scene Understanding and real-time passthrough capabilities.

By integrating directly with the physical room, AI enemies use actual furniture and walls for cover. The engineering focus was on real-time spatial analysis and dynamic AI positioning to ensure immersion regardless of room layout.

Core Experience

The experience is built on a "living room" architecture where the game state is managed by a spatial registry of detected real-world surfaces.

Spatial Awareness

Adaptive gameplay based on room mesh.

AI Interaction

Enemies respond to environmental changes.
