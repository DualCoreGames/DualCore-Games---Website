Project Overview

Mayaaverse is a standalone VR application built for Meta Quest and published on the Meta Store. It functions as a modular experience platform where users transition between cultural, wellness, and practical interaction modules.

The project’s engineering focus was on creating a cohesive navigation layer that manages state transitions between distinct 3D environments, including an AI-powered real estate hub with conversational interfaces.

Core Experience

The experience is divided into three specialized modules, each utilizing unique spatial audio and lighting configurations to define intent.
