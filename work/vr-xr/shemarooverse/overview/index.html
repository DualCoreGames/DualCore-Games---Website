Project Overview

ShemarooVerse is a cross-platform VR media ecosystem developed for Shemaroo Entertainment. It features custom theater environments, real-time media streaming, and integration with the JioImmerse platform.

The project required building a highly scalable content delivery system that supports low-latency video playback within a shared 3D social space, linking mobile ecosystems with high-end VR hardware.

Core Experience

The platform is designed around "Social Theaters" where multiple users can consume media simultaneously with synchronized playback and voice interaction.

Media Pipeline

Optimized low-latency video streaming.

Social Interaction

Multiplayer sync and voice chat.

Platforms

  • Android VR (JioGlass)
  • Meta Quest 2/3/Pro
  • iOS / Android Mobile
  • JioImmerse Ecosystem
