Project Overview

Never Open The Door is a psychological horror experience engineered around environmental storytelling, controlled pacing, and narrative-driven system design.

Instead of relying on constant scripted scares, the game uses a system-driven approach where narrative discovery directly influences environmental behavior, creating a living, reactive space that evolves based on player actions.

Core Experience

The experience is built on four distinct phases of psychological escalation, each managed by a central narrative state controller.

Exploration

Open-ended systemic investigation.

Distortion

Reactive reality destabilization.

Key Performance

  • System-driven pacing
  • Dynamic state controller
  • Zero-timer encounters
  • Measurable terror metrics

Gameplay Loop / Experience Flow

01

Discovery: Player uncovers narrative fragments and environmental clues.

02

Trigger: Discovery flags update the global narrative state machine.

03

Shift: The environment transforms dynamically to match the new state.
