Project Overview

Paapi Gudia is a cross-platform horror shooter designed to deliver intensive combat and atmospheric tension while maintaining hardware stability on both PCs and Meta Quest headsets.

The core engineering challenge was building a unified interaction system that feels native to both motion controllers and traditional input schemes while strictly optimizing vertex counts and draw calls for standalone VR performance.

Core Experience

The gameplay is structured around escalating threat waves and non-linear zone exploration, managed by a dynamic pacing engine.
