01 / Hybrid Development

One Engine. Two Realities.

Paapi Gudia is a cross-platform first-person horror shooter inspired by the iconic Bollywood horror film. The game delivers an immersive supernatural experience across two distinct hardware targets — Windows PC and Meta Quest standalone — from a single unified Unity codebase.

The central engineering challenge was delivering a genuinely immersive, atmospheric horror experience across platforms with vastly different rendering budgets — from a high-end PC GPU down to the Snapdragon XR2 mobile processor.

03 / System Architecture

Platform-Agnostic Core.

Built on a modular architecture where core gameplay is platform-independent. Specific behavior — rendering quality, input handling, locomotion — is injected at runtime through abstraction layers.

  • Unified InputRouted through a single InputActionAsset with XR, PC, and Gamepad schemes.
  • Runtime URP SwitchingDetects hardware and selects optimized PC or Quest render profiles.
  • Hybrid AIStandard NavMesh for minions, custom teleport graphs for the boss.
  • Tension ScalarA persistent scene manager updating dread levels based on real-time state.
