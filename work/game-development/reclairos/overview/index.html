GAMEPLAY – WAVE DEFENSE

Project Overview

Reclairos is a system-driven mobile strategy game built around escort-based combat and narrative progression.

Instead of defending a static base, players manage a moving objective — requiring continuous repositioning, unit coordination, and real-time threat prioritization.

Each encounter is constructed through modular gameplay systems, enabling scalable enemy behaviors, dynamic encounter variation, and stable performance across mobile hardware.

What Makes This Different

Escort-Based Combat

A moving objective transforms traditional tower defense into a dynamic system.

Class-Based Strategy

Units are built around functional roles requiring composition-based decisions.

Modular Encounters

Encounters are generated from data-driven systems, not hardcoded logic.

Mobile-First

Built for stable performance across mid-range mobile devices.
