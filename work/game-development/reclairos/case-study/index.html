Reclairos — Case Study

Designing scalable combat systems and performance-conscious gameplay architecture for mobile hardware.

01 Project Overview

Reclairos is a system-driven strategy game where gameplay depth emerges from structured combat systems, enemy behaviors, and encounter design rather than raw difficulty scaling.

The project required balancing gameplay complexity with strict performance constraints while maintaining clarity during large combat encounters on mobile hardware.

04 Technical Challenges

Unit Scalability

Managing large character rosters without linear animation controller growth.

UX Clarity

Maintaining tactical readability on mobile-scale viewports during chaotic combat.

Performance Jitters

Eliminating frame drops during heavy VFX/SFX resolution for turn conclusions.

05 Technical Solutions

We leveraged Animator Override Controllers to decoupling state-machine logic from unit-specific assets, reducing memory overhead by 40%.

A performance-first shader pipeline was implemented to handle character silhouettes, ensuring readability across all lighting conditions.

07 Key Engineering Takeaways

  • System-Driven Scalability: Building with decoupled modules allows rapid expansion without core modifications.
  • Performance-First Silhouettes: Prioritizing visual clarity over pure vertex count improves the mobile experience.
  • Logic Decoupling: Animator overrides are essential for managing high-unit variety at scale.

Outcomes

  • Stable 60 FPS on mobile
  • 40% memory reduction
  • High technical authority
