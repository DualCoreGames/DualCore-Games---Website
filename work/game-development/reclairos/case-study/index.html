01 Project Overview

Reclairos is a system-driven strategy game built around escort-based combat and narrative progression. The project required balancing complex gameplay systems with strict mobile performance constraints.

The challenge was to engineer a modular architecture that could handle dynamic enemy behaviors and real-time pathfinding for the moving objective while maintaining steady frame rates on mid-range hardware.

06 Architecture Principles

Modular Systems

Decoupled logic for rapid iteration and scalable content updates.

Encounter-Driven

Data-driven combat scenarios that adapt to player progression.

Scalable Asset Pipelines

Built for production-ready pipelines and cross-platform stability.

07 Challenges

Dynamic Pathfinding

Continuous recomputation of unit lanes as the objective moves.

Performance Spikes

Managing high-density VFX costs during lane-clearing resets.

Memory Constraints

Managing asset pools for 40+ unit types on 4GB RAM devices.

08 Solutions

Animator Overrides

Decoupling logic from assets, reducing controller overhead by 40%.

Object Pooling

Global singleton patterns for unit and VFX lifecycle management.

Culling Systems

Distance-based resolution and silhouette prioritizing.
