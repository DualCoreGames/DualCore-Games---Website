Overview

Crazy Planet is a 2D side-scrolling platformer designed around accessibility, responsiveness, and modular gameplay systems.

Inspired by classic arcade platformers, the project focuses on delivering fast, readable gameplay while maintaining a smooth progression curve. The game is built on a system-driven approach where level design, enemy behavior, and progression systems work together to create engaging gameplay through simplicity and clarity.

The result is a lightweight platformer experience that prioritizes performance, readability, and scalable content design.
