Reclairos: High-Fidelity Mobile Rendering Benchmark

A technical showcase demonstrating the potential of high-end visuals on consumer mobile hardware through specialized systems architecture and custom engine optimization.

Reclairos Technical Showcase Visual

The Challenge

Modern mobile devices have significant GPU overhead, yet many mobile games struggle to deliver consistent high-fidelity visuals without catastrophic thermal throttling or battery drain. DualCore Games set out to build an original technical benchmark that pushed the URP pipeline to its absolute limits while maintaining a stable 60 FPS on mid-range Android and iOS devices.

The Solution

By leveraging a custom rendering stack and procedural environment systems, we were able to shift significant computation from real-time GPU calculation to optimized CPU-bound systems. We implemented a custom LOD (Level of Detail) system that handled thousands of dynamic objects with minimal draw calls, alongside specialized shaders designed for the specific tiled-based rendering architecture of mobile GPUs.

Project Details

  • Platform: iOS, Android
  • Engine: Unity (Custom URP)
  • Service: Game Development
  • Category: Game Development Portfolio
  • Status: Technical Benchmark
  • Focus: Rendering & Optimization

Tech Applied

Custom HLSL Shaders Procedural Meshing Compute Shader Clusters GPU Instancing

Technical Breakdown

Systems Architecture

Built a modular data-oriented framework to handle complex environment state without the overhead of heavy GameObjects.

Rendering Pipeline

Stripped the standard URP to its essentials, injecting custom lighting passes that simulate global illumination on low-power hardware.

Optimization

Achieved a 40% reduction in draw calls through aggressive batching and custom GPU-side culling systems.

The Result

Reclairos stands as a core testament to DualCore's engineering capabilities. It has served as the technical foundation for several commercial collaborations, proving that studio-level visuals are achievable on mobile when approached with a systems-first mindset.
